# PDF-to-MCQ Generator - PDGQG 

## Overview

A full-stack web app that allows users to upload a syllabus (PDF), processes it, generates MCQs using an AI model, and presents a quiz interface with answers available at the end.

---
## Tech stack
- Nodejs
- Expressjs
- Nextjs
- PostgreSQL 

---
## API Spec
1. Upload PDF
2. Generate MCQs
3. Save Quiz
4. Get Saved Quiz
